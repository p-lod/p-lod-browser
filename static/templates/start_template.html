About P-LOD and About This Site

Pompeii Linked Open Data (P-LOD) is collecting and publishing data about Pompeii and its immediate environs as Linked Open Data. The project is jointly directed by Sebastian Heath (NYU) and Eric Poehler (UMass Amherst). It includes data collected for the Getty Foundation funded Pompeii Artistic Landscape Project (PALP) and the NEH funded Pompeii Bibliography and Mapping Project (PBMP). Data entry is ongoing and the content here will frequently change.

This site provides access through that data. Using the terminology of RDF, P-LOD uses URN-based identifiers and all resources are browseable via URLs with the pattern http://p-lod.org/urn/urn:p-lod:id:pompeii. There is also an API, which is growing and which will be documented by example for the time being. Each page that shows the information associated with a URN also creates links into the API. P-LOD is part of a network of sites that use the same LOD backend store. Currently, this site makes links to PALP.
